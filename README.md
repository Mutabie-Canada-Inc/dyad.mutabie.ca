## About This Repository

This repository contains the source code for the Dyad Framework landing page, a modern web application built with React, TypeScript, Tailwind CSS, and Vercel. It serves as both a showcase of our capabilities and a testing ground for experimental features.

## Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/dyad.mutabie.ca.git

# Navigate to the project directory
cd dyad.mutabie.ca

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) to view the application.

## Deployment

This application is configured for deployment to Vercel:

```bash
# Build the application
npm run build

# Deploy to Vercel
vercel --prod
```
